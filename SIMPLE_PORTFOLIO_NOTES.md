# Simple Portfolio Revision

This revision intentionally tones down the first v2 redesign.

## Changes

- Removed the heavy dark gradients and oversized marketing layout.
- Switched to a clean light theme with restrained blue accents.
- Reduced the number and visual weight of metrics.
- Kept the backend engineering positioning, but made the tone more understated.
- Preserved the important career signals: 50M+ users, ~1M requests/day, and 15s → <500ms P95 improvement.

## Build

```bash
npm install
npm run build
```
